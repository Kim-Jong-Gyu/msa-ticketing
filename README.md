## 🥅 프로젝트 목표
- MSA 구조가 무엇인지 학습한다.
- 리액티브 프로그래밍을 학습하고 적용한다.

## 📝 테스트 방법
```bash
1. 실행
./test-em-all.bash start
2. 작동 여부
./test-em-all.bash start stop
```
